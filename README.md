# Walmart Retail Operations Analytics (SQL + Python)

## Overview
End-to-end retail operations analytics project simulating Walmart store performance analysis.
Focus areas: revenue KPIs, inventory efficiency, stockouts, and seasonality.

## Tech Stack
- SQL
- Python (Pandas, NumPy, Matplotlib)
- GitHub

## KPIs
- Total Revenue KPI
- Stockout Rate KPI
- Inventory Turnover KPI
- Average Order Value KPI

## How to Run
1. Run `python/data_generator.py`
2. Execute SQL files in `sql/`
3. Run Python analysis scripts
